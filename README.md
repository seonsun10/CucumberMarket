# **CucumberMarket 오이마켓** - 오늘도 이용하는 마켓

## 구성원 및 역할 +++++각자 편집 ㄱㄱ
* 김동영 : 알림 관련 기능 및 페이지
* 김동훈 : 상품 관리 기능 및 페이지
* 박선우 : 상품 관리 기능 및 페이지
* 안경진 : 지역별 커뮤니티 게시판, 1:1 채팅 기능 및 페이지 
* 이종석 : 팀장. 회원 관련 기능 및 페이지, 협업 관련 기록 관리
* 이주호 : 1:1문의, 신고, 커뮤니티 게시판, FAQ

## 사용 기술
### 
* Java 11
* Spring Boot 2.7.2, Maven
  - Security, SMTP, STOMP
* JPA, Spring data JPA, Querydsl
* MyBatis
* JSP
* Oracle DB
* Ajax
* STOMP, SockJS
